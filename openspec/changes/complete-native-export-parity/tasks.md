# Полная совместимость экспорта УТ и БСП — план реализации

> **Для субагента:** используй
> `/subagent-dev openspec/changes/complete-native-export-parity/tasks.md`.
> Выполняй задачи по порядку зависимостей; независимые потоки A, B и C можно
> запускать параллельно с раздельным владением файлами.

**Цель:** получить воспроизводимую матрицу каждого файла на УТ и БСП, устранить
общие причины расхождений и достичь пустого raw-diff в обоих направлениях.

**Дизайн:** `openspec/changes/complete-native-export-parity/design.md`

**Базовый SHA:** фиксируется исполнителем непосредственно перед первой правкой.

**Порядок выполнения:** задачи 1–15 создают измерительный контур; задачи 16–25
закрывают первые доказанные классы дефектов; задачи 26–30 завершают полный цикл.

**Обозначения статуса:** `[x]` — выполнено и проверено; `[~]` — рабочая часть
реализована, но критерий задачи закрыт не полностью; `[ ]` — не начато.
